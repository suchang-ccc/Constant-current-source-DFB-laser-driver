# Constant-current-source-DFB-laser-driver
本设计采用DAC+运放构成Howland恒流源输出，实时采集输出电流与激光器内部TEC进行反馈调节，以达成数控恒流恒温驱动DFB蝶形激光器。
恒流源：使用14位IDAC+IV转换电路+跟随器+Howland运放电流源构成数控恒流输出。
采集、反馈：使用INA282电流检测ic配合20毫欧采样电阻进行输出电流采样，经过24位ADC转换后反馈至单片机。
控制：主控采用STM32通过软件SPI控制外部的ADDA模块，监测输出电流实现硬件和数控闭环，然后通过PID算法提高系统的响应速度和负载调整率。
指标：10-500mA步进1mA连续电流输出、电流精度0.5mA、负载调整率5%
